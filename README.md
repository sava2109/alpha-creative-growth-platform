# Alpha Creative — Digital Growth Platform

Professional multi-page agency website for Alpha Creative, focused on inbound marketing, HubSpot implementation, marketing automation, SEO, Google Ads, and custom web development.

## Overview
This project presents a commercial digital agency website designed to communicate service credibility, generate leads, and support business growth through a clean multi-page structure.

The website includes dedicated pages for:
- HubSpot setup and onboarding
- HubSpot marketing automation
- Inbound marketing campaigns
- Website design and development
- E-commerce development
- SEO services
- Google Ads services
- Pricing and company information

## Positioning
This repository is suitable for portfolio, freelance, and CV presentation because it represents a real-world business website with clear commercial intent, conversion-oriented page structure, and service-focused messaging.

## Deploy to GitHub Pages
This repository includes a GitHub Pages deployment workflow.

### Steps
1. Create a new GitHub repository named **alpha-creative-growth-platform**.
2. Push this project to the `main` branch.
3. In the GitHub repository, open **Settings > Pages**.
4. Under **Build and deployment**, select **GitHub Actions** as the source.
5. After the first push, GitHub will automatically publish the site.
