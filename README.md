# Alpha Creative — Digital Growth Platform

Multi-page agency website for Alpha Creative, focused on inbound marketing, HubSpot implementation, marketing automation, SEO, Google Ads, and custom web development.

## Overview
This project presents a commercial digital agency website built to communicate service credibility, generate leads, and support business growth through a clear multi-page structure.

Included pages and service areas:
- HubSpot setup and onboarding
- HubSpot marketing automation
- Inbound marketing campaigns
- Website design and development
- E-commerce development
- SEO services
- Google Ads services
- Pricing and company information

## Live Website
- Live preview: https://sava2109.github.io/alpha-creative-growth-platform/
- Repository: https://github.com/sava2109/alpha-creative-growth-platform

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Webflow-exported frontend structure
- Calendly integration
- Responsive multi-page layout

## Project Structure
- Home page: index.html
- About page: about-us.html
- Pricing page: pricing.html
- Service pages: root-level HTML files
- Styles: style.css
- Scripts: script.js

## Local Preview
Open `index.html` directly in a browser or preview the project from VS Code.

The repository includes a GitHub Pages workflow that deploys the site from the `main` branch through GitHub Actions.
